# Sudoku

Use any compiler to compile this c++ code, and execute it.

Usage:
Enter a Sudoku graph, it will print all possible solution

e.g.  
Input:  
0 0 1 2 0 0 3 0 0   
0 4 0 0 5 0 0 2 0  
6 0 0 0 0 4 0 0 1  
3 0 0 0 0 0 6 0 0  
0 6 0 0 0 0 0 5 0  
0 0 7 0 0 0 0 0 2  
7 0 0 5 0 0 0 0 8  
0 8 0 0 1 0 0 6 0  
0 0 5 0 0 3 7 0 0
