# Sudoku
enter a Sudoku graph, it will print all possible solution

e.g.  
enter this:  
0 0 1 2 0 0 3 0 0   
0 4 0 0 5 0 0 2 0  
6 0 0 0 0 4 0 0 1  
3 0 0 0 0 0 6 0 0  
0 6 0 0 0 0 0 5 0  
0 0 7 0 0 0 0 0 2  
7 0 0 5 0 0 0 0 8  
0 8 0 0 1 0 0 6 0  
0 0 5 0 0 3 7 0 0  
  
it will print:  
Solution 1:  
8 7 1 | 2 9 6 | 3 4 5  
9 4 3 | 1 5 7 | 8 2 6  
6 5 2 | 8 3 4 | 9 7 1  
-----+-------+------  
3 1 4 | 9 2 5 | 6 8 7  
2 6 8 | 3 7 1 | 4 5 9  
5 9 7 | 4 6 8 | 1 3 2  
-----+-------+------  
7 3 6 | 5 4 9 | 2 1 8  
4 8 9 | 7 1 2 | 5 6 3  
1 2 5 | 6 8 3 | 7 9 4  

//this sudoku only has 1 solution.
